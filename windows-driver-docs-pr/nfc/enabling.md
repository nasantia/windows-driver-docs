# Enabling NFP


A client can re-enable previously disabled subscriptions, publications, and presence with [**IOCTL\_NFP\_ENABLE**](https://docs.microsoft.com/windows-hardware/drivers/ddi/nfpdev/ni-nfpdev-ioctl_nfp_enable).

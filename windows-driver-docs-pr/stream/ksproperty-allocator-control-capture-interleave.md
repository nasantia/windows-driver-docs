# KSPROPERTY\_ALLOCATOR\_CONTROL\_CAPTURE\_INTERLEAVE


The KSPROPERTY\_ALLOCATOR\_CONTROL\_CAPTURE\_INTERLEAVE property informs the Overlay Mixer if interleaved capture is possible.

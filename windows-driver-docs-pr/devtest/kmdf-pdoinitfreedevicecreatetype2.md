# PdoInitFreeDeviceCreateType2 rule (kmdf)


The PdoInitFreeDeviceCreateType2 rule specifies that a driver must not call [**WdfDeviceCreate**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdevicecreate) after it calls [**WdfDeviceInitFree**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdfdevice/nf-wdfdevice-wdfdeviceinitfree).

|              |      |
|--------------|------|
| Driver model | KMDF |

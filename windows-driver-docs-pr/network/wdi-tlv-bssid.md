# WDI\_TLV\_BSSID


WDI\_TLV\_BSSID is a TLV that contains the BSSID of a BSS.

## TLV Type


0x2

## Length


The size (in bytes) of a [**WDI\_MAC\_ADDRESS**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address) structure.

## Values


| Type                                              | Description                                 |
|---------------------------------------------------|---------------------------------------------|
| [**WDI\_MAC\_ADDRESS**](https://docs.microsoft.com/windows-hardware/drivers/ddi/dot11wdi/ns-dot11wdi-_wdi_mac_address) | A Wi-Fi MAC address that specifies a BSSID. |

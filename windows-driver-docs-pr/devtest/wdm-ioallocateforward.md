# IoAllocateForward rule (wdm)


The **IoAllocateForward** rule specifies that if an IRP is generated by a call to [**IoAllocateIrp**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-ioallocateirp), the driver must set a completion routine before calling [**IoCallDriver**](https://docs.microsoft.com/windows-hardware/drivers/ddi/wdm/nf-wdm-iocalldriver) or [**PoCallDriver**](https://docs.microsoft.com/windows-hardware/drivers/ddi/ntifs/nf-ntifs-pocalldriver).

|              |     |
|--------------|-----|
| Driver model | WDM |
